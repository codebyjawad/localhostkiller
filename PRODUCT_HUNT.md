# LocalhostKiller 🔥

**Tagline:** Kill localhost processes with style – A beautiful CLI for developers

**Description:**
Ever had "port 3000 is already in use" errors? LocalhostKiller shows you what's running on localhost and lets you kill processes instantly.

**Features:**
- 🎯 See all localhost ports at a glance
- ⚡ Kill processes with one command
- 🎨 Beautiful terminal UI
- 🔄 Interactive mode
- 💻 Cross-platform

**Installation:**
```bash
pip install localhostkiller
```

**Usage:**
```bash
lhk              # Interactive mode
lhk 3000         # Kill port 3000
lhk --list       # List all ports
```

**Why it's useful:**
- No more port conflicts
- No more remembering complex lsof commands
- See all your dev servers at once
- Beautiful, user-friendly interface

**Tech Stack:**
Python, Rich, psutil
