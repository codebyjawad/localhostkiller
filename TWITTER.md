I built LocalhostKiller 🔥 – a CLI tool to see and kill localhost processes instantly

No more "port 3000 is already in use" errors
No more `lsof -ti:3000 | xargs kill -9`

Just: `lhk 3000` ⚡

Features:
• Interactive mode with menu
• Beautiful terminal UI
• Kill by port in one command
• Cross-platform (Mac/Linux/Windows)

Install: pip install localhostkiller

GitHub: https://github.com/jawad/localhostkiller
