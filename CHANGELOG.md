# Changelog

All notable changes to LocalhostKiller will be documented here.

## [0.1.0] - 2026-03-20

### Added
- Initial release
- Interactive mode with menu selection
- Kill by port: `lhk 3000`
- List all localhost processes: `lhk --list`
- Beautiful terminal UI with Rich
- Kill all option
- Confirmation prompts
- Help and version flags
- Cross-platform support (Mac, Linux, Windows)

### Features
- Port scanning and process identification
- One-command process killing
- Real-time process display
- Colorful, user-friendly interface
